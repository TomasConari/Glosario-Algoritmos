Glosario

Algoritmo

Algoritmo

Secuencia ordenada de pasos que resuelve un problema o realiza una tarea.

Variable

Variable

Espacio en memoria para almacenar datos que pueden cambiar durante la ejecución de un programa.

Condicional

Condicional

Instrucción que ejecuta acciones dependiendo de si una condición es verdadera o falsa.

Iteración

Iteración

Proceso de repetir una serie de instrucciones hasta cumplir una condición.

Diagrama de Flujo

Diagrama de Flujo

Representación gráfica de un algoritmo mediante símbolos conectados por flechas.

Pseudocodigo

Pseudocodigo

Descripción simplificada de un algoritmo utilizando lenguaje natural y términos de programación.

Entrada

Entrada

Datos iniciales que recibe un algoritmo para procesar y generar resultados.

Salida

Salida

Resultados generados por un algoritmo tras procesar las entradas.

Bucle

Bucle

Instrucción que permite repetir bloques de código varias veces.

Eficiencia

Eficiencia

Medida del uso óptimo de recursos (tiempo y espacio) al ejecutar un algoritmo.
